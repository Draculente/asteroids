About

Welcome to Asteroids, a web-based game developed as a part of the Web and Cloud Computing Project in the year 2023 at the Technical University of Lübeck.

Game Objective: Asteroids is a classic arcade-style space shooter game where the player controls a spaceship and navigates through a dangerous asteroid field. The objective is to survive as long as possible by shooting down and avoiding collisions with the incoming asteroids.

Technologies Used: The development of Asteroids involved a combination of multiple programming languages and frameworks to create a seamless and enjoyable gaming experience.

  • Backend: Python - Python was utilized for the game's backend, ensuring efficient handling of various game-related functions and operations.
  • Game Logic: Rust (WebAssembly) - The game logic was implemented in Rust using WebAssembly, enabling high-performance execution of the game mechanics and operations.
  • View and API integration: Typescript - The game's user interface and API integration were implemented using Typescript, providing an interactive and responsive front-end experience.
